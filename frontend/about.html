About the System

The Internal Marks Management and Analysis System is designed to help institutions efficiently manage and analyze student internal assessment data.

This system allows faculty to enter marks, track student performance, and generate insights based on academic data.

It provides a structured and user-friendly interface for storing, retrieving, and analyzing internal marks across semesters and subjects.

Key Features

✔ Easy mark entry (subject-wise)

✔ Performance analysis (average & grading)

✔ Student-wise data tracking

✔ Dynamic internal assessment support
